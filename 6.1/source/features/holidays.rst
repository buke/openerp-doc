Holidays Management
===================

This module's main task is to control the holiday's schedule of your company.
It allow employees to requests holidays. Then, managers can review requests for
holiday, and control the overall holiday's planning for the company or department.

.. raw:: html
 
 <a target="_blank" href="../images/holidays_screenshot.png"><img src="../images_small/holidays_screenshot.png" class="screenshot" /></a> 

Key points:
-----------

* Validation system included
* Customisable workflow for validation
* Support all kinds of leaves (sickness, holidays, paid days, ...)
* Calendar view

Integration benefits:
---------------------

* Integrated to the company calendar
* Integrated with the payroll system for automatic computation
