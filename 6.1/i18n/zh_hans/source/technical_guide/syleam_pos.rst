.. i18n: Description

Description
-----------

.. i18n: ::
.. i18n: 
.. i18n:   
.. i18n:   Main features :
.. i18n:    - Fast encoding of the sale.
.. i18n:    - Allow to choose one payment mode (the quick way) or to split the payment between several payment mode.
.. i18n:    - Computation of the amount of money to return.
