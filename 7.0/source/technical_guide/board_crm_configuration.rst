Description
-----------

::

  This module implements a dashboard for CRM that includes:
      * My Leads (list)
      * Leads by Stage (graph)
      * My Meetings (list)
      * Sales Pipeline by Stage (graph)
      * My Cases (list)
      * Jobs Tracking (graph)
