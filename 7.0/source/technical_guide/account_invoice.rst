Description
-----------

::

  India Accounting module includes all the basic requirements of
      Basic Accounting, plus new things which available are
      * Indian Account Chart
      * New Invoice - (Local, Retail)
      * Invoice Report
      * Tax structure
      * Journals
      * VAT Declaration report
      * Accounting Periods
